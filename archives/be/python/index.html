# 型別

目前已知有以下型別：

  1. 字串

# 注意

  1. 複數
  2. 布林值
  3. 列表
#基本存值

#清除多欄位內容：以下面來說就是清除 inde 0 - 3 都是 []
data[0:3]=[]

#從頭加入: insert(index,value)
data.insert(0,4)

#取得列表總長
len(data)
  1. tuple

# 流程控制

最常見的：

  1. if else
  2. 三元判斷
  3. 多個條件同時成立時 and
  4. 其一成立即可 or
  5. 不是時成立 not
# if .. else
data = true
if data:
   print('is true!')
else:
   print('not! is false')

# 三元


# and
if a and b:
   print('both success!')

# or
if a or b:
   print('one is true!')

# not
if not a:
   print('not !')
